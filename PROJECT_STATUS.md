## 技术栈

### 后端
- **框架**: FastAPI 0.110+
- **ORM**: SQLAlchemy 2.0 (异步)
- **数据库**: PostgreSQL 15+
- **缓存**: Redis 7
- **认证**: JWT (python-jose)
- **密码**: SHA256 + salt
- **迁移**: Alembic
- **文件存储**: MinIO
- **任务队列**: Celery (已配置)

### 前端
- **框架**: React 18 + TypeScript 5
- **UI库**: Ant Design 5 + Pro Components
- **状态**: Zustand (持久化)
- **数据**: TanStack Query (React Query)
- **路由**: React Router 6
- **构建**: Vite 5
- **图表**: @ant-design/charts

## 项目结构

```
safety-service-system/
├── backend/              # 后端项目
│   ├── app/
│   │   ├── api/v1/      # API路由 (13个模块)
│   │   ├── core/        # 核心功能 (安全、权限、中间件)
│   │   ├── crud/        # 数据库操作
│   │   ├── models/      # 数据模型 (11个模型文件)
│   │   ├── schemas/     # Pydantic模型
│   │   └── services/    # 业务服务 (MinIO等)
│   ├── alembic/         # 数据库迁移
│   ├── scripts/         # 初始化脚本
│   └── Dockerfile
├── frontend/             # 前端项目
│   ├── src/
│   │   ├── api/         # API请求 (11个模块)
│   │   ├── components/  # 组件
│   │   ├── pages/       # 页面 (10+模块)
│   │   ├── stores/      # 状态管理
│   │   └── types/       # TypeScript类型
│   └── Dockerfile
├── docker-compose.yml    # 容器编排
├── dev-start.sh          # 开发启动脚本
├── manage-infra.sh       # 基础设施管理
└── README.md
```

## 启动方式

### 方式1: 一键启动（推荐开发）

```bash
# 启动基础设施
./manage-infra.sh start

# 启动前后端
./dev-start.sh
```

### 方式2: Docker Compose

```bash
docker-compose up -d
```

### 方式3: 手动启动

```bash
# 后端
cd backend && uvicorn app.main:app --reload

# 前端
cd frontend && npm run dev
```
